## 启动python
三种启动python的方案：创建环境后运行代码、在编译器中运行、终端直接运行代码文件。

### 创建环境后运行代码
```powershell
python  # 通过该命令进入python环境
>>>     # 出现">>>"符合时，代表已经进入python环境
```
通过ctrl+Z可以退出python环境，回到命令行中。

### 在编译器中运行
在VScode中可以运行.py文件。

### 终端直接运行代码文件
```powershell
python <.py文件>
```
通过该命令可以运行对应的.py文件。

## 变量名规则
- 只包含字幕、数值和下划线，且不能以数字为首；
- 不能包含空格；
- 不能将关键字和函数名作为变量名；
- 尽量使用小写字母为变量名；
- 慎重小写字母l和大写字母O 。

### 字符串
字符串的定义如下：
```python
' 单引号字符串 '
" 双引号字符串 "     # 以上两种引号所阔内容均属于字符串
" 能够在'双引号'内显示单引号 "  # 使用不同引号来让引号也属于字符串是允许的
```

#### 修改字符串格式的方法类型
```python
message = "Hello python "
print(message.<方法method>())
```
通过<方法method>，可以对字符串内数据进行修改后再输出，格式`<变量名>.<方法>()`，"."表示使用该方法，"()"是必须的没有内容只是因为他的操作不需要输入。
- 有关大小写的方法
```python
title()     # 全部首字母大写
upper()     # 所有字母大写
lower()     # 所有字母小写
```
无论处理前数据大小写如何，处理后均按<方法>对应显示。
- 删除字符串内无用的空白
```python
rstrip()     # 右端空格清除
lstrip()     # 左端空格清除
strip()      # 左右端空格均清除
```
方法只是对输出的格式进行修改，并不会改变原变量中的字符串值。要清除原值参考[删除空白](#永久删除字符串中空白)。

#### f字符串--变量组成字符串
```python
f"{<变量名1>} {<变量名2>}"                  # 将不同的变量组成一个字符串
f"abcd,{<变量名1>} {<变量名2>}"             # 将字符与变量组成一个字符串
message = f"abcd,{<变量名1>} {<变量名2>}"   # 将f字符串赋给变量是允许的
```
f字符串（format），能够在一个字符串中利用变量，将变量组成字符串。

#### 制表符与换行符
```python
"\t"    # 制表符
"\n"    # 换行符
```

#### 永久删除字符串中空白
```python
message = " python "
message = message.strip()   # 将删除空白后的字符串结果赋值到原变量中
print(message)              # 该输出将是删除空格后的版本，且message变量中的字符串已经是没有空格的版本了。
```
